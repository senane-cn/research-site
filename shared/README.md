# Quarterly Sharing 页面维护说明

这个文件夹对应页面：

```text
https://research.conservision.com/shared/
```

请注意：

- `index.html` 是 Quarterly Sharing 页面，可以在这里更新资料分享内容。
- `research.css` 是整个 research 网站共用样式，请不要删除或随意改名。
- 如果未来有资料文件，建议新建 `data/` 或 `assets/` 文件夹来存放。

推荐结构：

```text
shared/
  index.html
  research.css
  README.md
  data/
  assets/
```
